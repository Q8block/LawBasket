Installation

We have used HTML, CSS, and JS to develop the template. There is nothing to install. Just unzip the template zip folder then replace demo contents and use it. Enjoy!

Configuration

We have used HTML, CSS, and JS to develop the template. There is nothing to configure.

How to Build for Production?

Quiety is an HTML template so no need to build. Just replace the demo contents with your company contents then upload the main file and folder under the Quiety folder in your server root folder.

How to make HTML template live?

Quiety is an HTML template so you need to change the template contents and images with your company information and images in an HTML file.

If you need to change or update any CSS then go to assets>css>custom.css (assets>css>main.css file in our template CSS file so no need to change it. If you need to change any CSS you should write custom.css file.) file updated or edit this file then upload the changed file to your live server. Same way If you need to change custom scripts then go to assets>js>app.js file.

Steps to make live:

  • Download your item from Envato and unzip it.
  • Add all files and folders under the Quiety folder to your hosting webroot.
  • Unzip your uploaded file if it is a zip version and browse from your domain: yourdomain.com

How to change images?

The downloaded package does not contain images that we are showing on the demo. You will find all images under assets>img/ folder. You can replace the placeholder image URL with your image URL like assets/img/your-image.png or img/screen/your-image.svg or img/blog/your-image.jpg

We have only included SVG images. It's MIT licenses Open-source illustrations for every project. You can use it.

Change background image

To change the background image, you need to update img>your image path.png. When you change the image name then you should update the image path on your style.css file.

.hero{
  background-image: url(../img/hero/hero-one.png);
}

Another way we have used an inline background image in section.

<section class="hero-section ptb-120 text-white bg-dark" style="background: url('img/hero-dot-bg.png')no-repeat center right">
